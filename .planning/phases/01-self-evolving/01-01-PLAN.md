<task type="auto">
  <name>Task 3: Add `skill` subcommand to piharness.sh</name>
  <files>
    piharness.sh
  </files>
  <action>
    Add a `skill` subcommand to piharness.sh with three verbs:

    ##### `skill list`
    Usage: `./piharness.sh skill list`
    Reads ~/.piharness/skills/registry.json and displays a formatted table:
    ```
    NAME                VERSION  USAGE  TRIGGERS            CREATED
    prompt-escaping     1        0      3                   <date>
    ```

    ##### `skill show <name>`
    Usage: `./piharness.sh skill show <name>`
    Shows the full SKILL.md content for the named skill.
    Looks up name in registry.json, reads the path/SKILL.md, prints it.
    Error if skill not found.

    ##### `skill new <name> [--pattern "trigger phrase"] [--desc "description"]`
    Usage: `./piharness.sh skill new my-skill --pattern "does X" --desc "Description"`
    Creates a new skill scaffold:
    1. Creates ~/.piharness/skills/<name>/
    2. Writes a template SKILL.md with the provided name/desc/patterns
    3. Creates an empty template.md
    4. Registers in registry.json
    5. Prints confirmation

    Implementation details:
    - Insert the `skill)` case block before the `help|*)` fallback (before "help|*)" line)
    - Use the same pattern as other subcommands: parse args with shift loop
    - SKILLS_DIR="$PIHARNESS_DIR/skills" (use PIHARNESS_DIR env var with default)
    - For registry reads: use python3 -c "import json,os; ..." to parse, or use jq if available
    - REGISTRY="$SKILLS_DIR/registry.json"
    - Use helper: _registry_read() { python3 -c "import json,sys; json.dump(json.load(sys.stdin), sys.stdout, indent=2)" < "$REGISTRY"; }
    - For simplicity and reliability, use python3 for JSON manipulation (more portable than jq)

    Code structure:
    ```bash
    skill)
      # skill list|show|new args...
      action="${1:-list}"; shift || true
      SKILLS_DIR="${PIHARNESS_DIR:-$HOME/.piharness}/skills"
      REGISTRY="$SKILLS_DIR/registry.json"
      case "$action" in
        list)
          if [[ ! -f "$REGISTRY" ]]; then echo "No skills."; exit 0; fi
          python3 -c "
    import json, os, sys
    reg = json.load(open(os.path.expanduser('$REGISTRY')))
    print('%-24s %-8s %-6s %-8s %s' % ('NAME','VERSION','USAGE','TRIGGERS','CREATED'))
    print('%-24s %-8s %-6s %-8s %s' % ('----','-------','-----','--------','-------'))
    for s in reg.get('skills', []):
        print('%-24s %-8d %-6d %-8d %s' % (s['name'], s.get('version',1), s.get('usage_count',0), len(s.get('trigger_patterns',[])), s.get('created','')))
    " || echo "Error reading registry"
          ;;
        show)
          name="$1"
          [[ -z "$name" ]] && { echo "Usage: piharness skill show <name>" >&2; exit 1; }
          python3 -c "
    import json, os
    reg = json.load(open(os.path.expanduser('$REGISTRY')))
    for s in reg['skills']:
        if s['name'] == '$name':
            skill_file = os.path.join(s['path'], 'SKILL.md')
            if os.path.exists(skill_file):
                print(open(skill_file).read())
            else:
                print('Skill found but SKILL.md missing at:', skill_file)
            exit(0)
    print('Skill \"$name\" not found')
    exit(1)
    " || true
          ;;
        new)
          name="$1"; shift
          pattern=""; desc=""
          while [[ $# -gt 0 ]]; do
            case "$1" in
              --pattern) pattern="$2"; shift 2 ;;
              --desc)    desc="$2";    shift 2 ;;
              *) echo "Unknown: $1" >&2; exit 1 ;;
            esac
          done
          [[ -z "$name" ]] && { echo "Usage: piharness skill new <name> [--pattern ...] [--desc ...]" >&2; exit 1; }
          skill_dir="$SKILLS_DIR/$name"
          mkdir -p "$skill_dir"
          cat > "$skill_dir/SKILL.md" <<SKILLEOF
    ---
    name: $name
    version: 1
    created: $(date -u +%Y-%m-%d)
    usage_count: 0
    trigger_patterns: [$(printf '"%s"' "$pattern")]
    description: "$desc"
    ---

    # Skill: $name
    SKILLEOF
          touch "$skill_dir/template.md"
          # Register
          python3 -c "
    import json, os
    reg_path = os.path.expanduser('$REGISTRY')
    reg = json.load(open(reg_path))
    reg['skills'].append({
        'name': '$name',
        'path': '${skill_dir}',
        'trigger_patterns': [$(printf '"%s"' "$pattern")],
        'description': '$desc',
        'usage_count': 0,
        'created': '$(date -u +%Y-%m-%d)',
        'version': 1,
        'auto_extracted': false,
        'template_file': 'template.md'
    })
    reg['last_updated'] = '$(date -u +%Y-%m-%dT%H:%M:%SZ)'
    json.dump(reg, open(reg_path, 'w'), indent=2)
    "
          echo "Created skill: $name"
          echo "  Edit: $skill_dir/SKILL.md"
          echo "  Template: $skill_dir/template.md"
          ;;
        *)
          echo "Usage: piharness skill list|show|new" >&2; exit 1
          ;;
      esac
      ;;
    ```

    Insert this block before the `help|*)` case. Update the help text to include the new commands.

    Also add this to the help section (~line 1009) under a new section:

    ```
      skill list                          List installed skills
      skill show <name>                   Show skill details
      skill new <name> [--pattern ...]    Create new skill scaffold
    ```
