PIHARNESS

Self-evolving AI worker orchestrator. Fallback across 7 runtimes, overnight skill extraction, role-based pipelines — all from your terminal.

Features

Everything you need to orchestrate AI coding agents at scale.

🔁

Multi-Runtime Fallback

Tasks fall through 7 AI backends when one hits rate limits. Intelligent cooldowns and best-runtime selection.

🧠

Self-Evolution

Watches task patterns and auto-extracts reusable skills from clusters of 3+ similar tasks. Runs overnight via nightly.

👷

Role-Based Pipeline

Spawne implementer, tester, and reviewer workers — run a full pipeline with a single command using pipeline.

🔄

Orchestrator Supervision

Long-running tasks auto-handoff to the next runtime when token limits are hit. State is preserved across handoffs.

📊

Usage Analytics

Per-runtime health dashboard: success rates, latency, last used. Data drives intelligent fallback decisions.

📦

Skill System

Capture patterns as reusable skills with trigger patterns, descriptions, and version tracking. List, show, and suggest.

Commands

Full command reference for PIHARNESS.

CommandDescription
spawn [--role R]Spawn a worker pane (optionally with a role)
task <s> <prompt>One-shot task on a specific worker
auto <s> <prompt>Task with automatic runtime fallback
supervise <task>Long-running orchestrator with auto-handoff
pipeline <task>Implement → test → review pipeline loop

Get Started

Clone the repo and run your first task in seconds. 

$ git clone https://github.com/RubberDucky3/PIHARNESS.git
$ cd PIHARNESS

# Spawn a worker and run a task
$ ./piharness.sh spawn --label my-worker
$ ./piharness.sh auto my-worker "Build a flappy bird game in Python"

# Run a pipeline
$ ./piharness.sh pipeline "Add user authentication" --rounds 3

# Overnight self-evolution
$ ./piharness.sh nightly
